# PHP Login-System
User Login and Registration system with Bootstrap

## Files
1. config.php - 
***This is the file that PHP uses to connect to the Database***
2. login.php - 
***This is the login form as well as the PHP script to check the username and passwords against what is stored in the database, and starts the session with unique session ids***
3. register.php - 
***This is the registration form along with the PHP script to do the form validation and insert the form data into the database***
4. welcome.php - 
***The Dashboard the user sees after successfully logging in***
5. reset-password.php -
***Reset Password form page with the PHP script to change it in the database***
6. logout.php - 
***Logs out the user by ending the session***
7. users.sql -
***The SQL Query to setup the database table***
